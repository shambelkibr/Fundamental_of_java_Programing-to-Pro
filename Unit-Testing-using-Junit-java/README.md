# 🧮 Calculator Unit Testing with JUnit

This repository contains a **Java calculator application** with comprehensive **unit tests using JUnit**.  
Each calculator operation is implemented as a separate module and thoroughly tested to ensure correctness and reliability.

---

## 📌 Project Overview

The goal of this project is to demonstrate:
- Modular Java design
- Unit testing best practices
- Validation of arithmetic operations using **JUnit**

Each arithmetic operation (addition, subtraction, multiplication, division, etc.) is tested independently.

---

## ✨ Features

✔ Modular calculator implementation  
✔ Unit tests for each operation  
✔ Edge case handling (zero, negative numbers, division by zero)  
✔ Clean and readable test structure  
✔ Built using **Java + JUnit**

---

## 🧩 Modules Tested

| Operation        | Description                          |
|------------------|--------------------------------------|
| Addition         | Tests sum of two or more numbers      |
| Subtraction      | Tests difference between numbers     |
| Multiplication  | Tests product of numbers             |
| Division         | Tests quotient and zero handling     |
| Modulus (Optional) | Tests remainder calculation       |

## 🛠 Technologies Used

- **Java**
